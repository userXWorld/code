# HIV治疗策略优化系统

## 项目简介

基于深度强化学习(DQN)的HIV长期治疗策略优化系统，提供个性化治疗方案生成和全面的模型评估框架。

## 文件说明

- **hiv_treatment_optimization.py**: 主优化系统，包含环境模拟、DQN智能体和训练流程
- **hiv_model_evaluation.py**: 评估框架，提供有效性、鲁棒性分析和改进建议
- **EVALUATION_USAGE.md**: 评估框架使用指南

## 快速开始

### 训练模型

```bash
python hiv_treatment_optimization.py
```

### 评估模型

```bash
python hiv_model_evaluation.py
```

详细使用方法请参考 [EVALUATION_USAGE.md](EVALUATION_USAGE.md)
